## Basic Elysia.js API With MongoDB Auth
Here is the basic Elysia.js API with MongoDB Auth. This is a simple API that uses MongoDB for authentication. This API has the following features:

- [x] User Registration
- [ ] User Login
- [ ] User Logout
- [ ] User Profile
- [ ] User Update
- [ ] User Delete
- [ ] User List
- [ ] Admin seed
- [ ] User approval
- [ ] User role
- [ ] User permission
- [ ] User password reset
- [ ] Email verification
- [ ] Email notification
